## Fast Workflow in Unity

1. Run the auto generator menu (see script below):
   - CAE -> B737 -> Generate Input Assets and Registry
2. Open generated assets under Assets/Data/B737/Inputs.
3. Adjust Display Name, AI Description, and value ranges where needed.
4. Add CockpitInputBinding on each mapped scene object and assign matching asset.
